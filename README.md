# Pyramid Lamp
Pyramid shaped lamp with 3W RGB and white LED versions.

## Specifications
- Diffuse light and project it to the ceiling.
- Originally printed in PLA.
- Two versions:
  - 3W RGB LED with PWM circuit.
  - 3W white LED (no PWM).
- UFO-shaped LED cover.
- Manually crafted heatsink for now. (A great upgrade would be an aluminum PCB)
- Small power supply housed within the base.
